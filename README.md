
# Student Portfolio – Collaborative Git & GitHub Project

This is a group learning project designed to practice Git and GitHub through real collaboration.  
Three members will build a simple portfolio website together using branches, pull requests, and code reviews.

---

## 1. Project Goal
- Understand Git basics through real hands-on work.
- Learn branching, merging, and resolving merge conflicts.
- Practice team collaboration using GitHub.
- Build a small working website as a final result.

---

## 2. Team Members
- **Member 1:** Repository setup, PR reviews, merges.
- **Member 2:** Portfolio section development.
- **Member 3:** Portfolio section + a small JavaScript feature.

---

## 3. Tools Required
- GitHub account  
- Git installed on computer  
- A code editor (VS Code recommended)  
- Web browser (Chrome recommended)

---

## 4. Project Folder Structure

student-portfolio/
│── README.md
│── index.html (to be added by members)
│── assets/ (images, CSS, JS files will go here)
